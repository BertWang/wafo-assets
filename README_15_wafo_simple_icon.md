# WAFO Simple Icon Expansion V1

本包用於擴充 `BertWang/wafo-assets`。

## Contents

- 98 icons
- SVG currentColor master
- SVG fixed WAFO gold `#B88A44`
- PNG black transparent 512
- PNG gold transparent 512
- CSS mask class
- Demo HTML
- Manifest CSV

## Upload target

將本資料夾內容合併到 repo 根目錄：

```text
assets/svg/15_wafo_simple_icon/
assets/svg/15_wafo_simple_icon_gold/
assets/png512/15_wafo_simple_icon/
assets/png512/15_wafo_simple_icon_gold/
css/wafo-simple-icons-v1.css
demo/wafo_simple_icon_demo.html
manifest/wafo_simple_icon_manifest_v1.csv
preview/preview_15_wafo_simple_icon.jpg
```
